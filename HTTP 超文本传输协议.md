HTTP

- HyperText Transfer Protocol 超文本传输协议

- 网络传输协议

- 应用层的协议

- HTTP 本身是无状态的，需要额外去实现状态的记录，比如 Cookie、Session 技术

- 客户端发起请求，服务端进行响应的形式

- 

- 基于 TCP/IP 协议传输数据，是可靠的传输协议

- 简单、灵活可扩展的

- 以明文形式传输数据

  

  

是一个基于请求与响应，无状态的，应用层的网络传输协议
